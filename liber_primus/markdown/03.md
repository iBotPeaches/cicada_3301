* Using a reverse Vigenere cipher with the key `divinity` (`ᛞᛁᚢᛁᚾᛁᛏᚣ`)
* Run the entire text as one continuous chunk through the cipher
* Skip the shift on the following runic indexes
  * 59, 95, 108, 170, 204, 205, 312
