---

### Method

* Reverse the runic decoding. (ie `F` -> `EA`, `EA` -> `F`)
* Shift +3 additionally (ie `NG [21]` + 3 = `A [24]`)

### Tool

```
➜ php cicada app:rotation-cipher --runic-shift --rotation=3
```
